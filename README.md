# Study Sphere
Study Sphere is a modern web application designed to help students organize their study materials, track progress, and improve productivity through a clean and intuitive interface.

---
## Features
- Create and manage study notes
- Organize content into subjects/modules
- Track learning progress
- Secure authentication system
- Simple and responsive UI
- Fast performance with Vite + React

---
## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js / Express 
- **Database:** PostgreSQL 
- **Authentication:** JWT / Session-based auth
- **Styling:** Tailwind CSS
